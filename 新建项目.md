# 创建新的项目

```
django-admin startproject projectname
```

# 创建新的APP

```
python3 manage.py startapp my_app
```

# 启动Django中的开发服务器

```python
# make new migrations
python3 manage.py makemigrations
 
# apply all migrations
python3 manage.py migrate
 
# run server
python3 manage.py runserver

```
